# Milestone 2 Project 1 - Personal Portfolio Website

## Project Overview

This is a **personal portfolio website** built with **HTML** and **custom CSS** as part of **Milestone 2**. The website showcases my skills, projects, and achievements as a **Web Developer**. The design follows featuring a professional layout with multiple sections, including an **About** section, **Skills** list, and a **Contact** form.

## Features

- **Responsive Design**: The website is fully responsive, with optimized views for desktop and mobile.
- **Custom CSS Styling**: All styling is handled using **custom CSS** for better performance and flexibility.
- **Profile Section**: Displaying personal details with a profile image.
- **Skills Section**: Showcasing key technical skills.
- **Contact Form**: Users can contact me directly via a form submission.

## Technologies Used

- **HTML**: The structure of the portfolio.
- **CSS**: Custom CSS for styling, including flexbox for layout and media queries for responsiveness.
